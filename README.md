# PTIT Code Helper

> Tiện ích Chrome hỗ trợ sinh viên sử dụng hệ thống PTIT Code thuận tiện hơn

## ✨ Tính năng

### 📝 Soạn thảo & Nộp bài
- **Code Editor tích hợp**: Gõ code trực tiếp với Ace Editor ngay trong trang bài tập
- **Chức năng bản nháp - Tự động lưu** bài đang soạn trên Editor. Cho phép tùy chỉnh chế độ và thời gian lưu.
- **Kéo-thả tệp** hoặc **Ctrl+C, Ctrl+V** để nhập code
- **Nộp bài siêu tốc**:
  - **Nộp trực tiếp** từ Editor
  - **Nộp từ bộ nhớ tạm** (Clipboard)

### 🔧 Tích hợp CPH (Competitive Programming Helper)
- **Tự động nhập bài vào CPH**
- **Tùy chỉnh tên file với nhiều định dạng:**
  - **Mã bài**: `CHELLO`, `chello`
  - **Tên bài**:  `IN HOA`, `in thường`, `camelCase`, `PascalCase`, `snake_case`

### 📥 Công cụ tiện ích
- **Sao chép đề bài và test cases**
- **Tải xuống đề bài**
- **Tìm bài giải mẫu trên GitHub**

### 🎨 Tùy chỉnh giao diện
- **Ẩn banner "Thử nghiệm phiên bản mới"**
- **Ẩn mục chat**
- **Ẩn danh sách bài giảng**

### 🔍 Lọc bài tập
- **Ẩn bài làm đúng**
- **Chỉ hiện bài làm sai**

## 📦 Cài đặt

1. Tải xuống từ **[trang của mình](https://anvu.web.app/pch/ptit-code-helper.zip)** hoặc **[GitHub Releases](https://github.com/anvu17/PTIT-Code-Helper/releases/)**
2. Giải nén tệp zip
3. Truy cập mục **Tiện ích mở rộng** (Extensions) trên Chrome: `chrome://extensions/`
4. Bật **"Chế độ nhà phát triển"** (Developer mode)
5. Nhấn vào **"Tải tiện ích đã giải nén"** (Load unpacked)
6. Chọn thư mục bạn vừa giải nén
